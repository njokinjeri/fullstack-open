## Course Overview

The course covers full-stack web development, with a focus on React, Redux, Node.js, MongoDB, GraphQL, and TypeScript.


## Course Parts

### Part 0: Fundamentals of Web Apps

- **Topics**: HTTP, HTML, CSS, JS basics, application architecture.
- **Key Project**: Diagramming a web app's lifecycle.

### Part 1: Introduction to React

- **Topics**: React components, props, state, event handling.
- **Key Project**: Course Information app.

### Part 2: Communicating with Server

- **Topics**: REST APIs, Promises, Axios, effect hooks.
- **Key Project**: Phonebook app.

### Part 3: Programming a Server with Node.js and Express

- **Topics**: Node.js, Express, middleware, MongoDB, error handling.
- **Key Project**: Phonebook backend.

### Part 4: Testing Express Servers, User Administration

- **Topics**: Unit testing, integration testing, Jest, supertest.
- **Key Project**: Blog list backend with user management.

### Part 5: Frontend with React and Redux

- **Topics**: React components, Redux basics, reducers, actions.
- **Key Project**: Blog list frontend.

### Part 6: State Management with Redux

- **Topics**: Advanced Redux, Thunk, selectors, state normalization.
- **Key Project**: Anecdote app with Redux state.

### Part 7: React Router, Custom Hooks, and Advanced Features

- **Topics**: React Router, useHistory, custom hooks, performance optimization.
- **Key Project**: Blog list with routing and advanced features.

### Part 8: GraphQL

- **Topics**: Apollo, GraphQL schema, queries, mutations, subscriptions.
- **Key Project**: GraphQL Book library app.

### Part 9: TypeScript

- **Topics**: TypeScript basics, interfaces, types, working with React and TypeScript.
- **Key Project**: TypeScript integration in a full-stack project.

### Part 11: Continuous Integration / Continuous Delivery

- **Topics**: CI/CD pipelines, automated testing, linting and deployment workflows
- **Key Project**: Implementing a full CI/CD pipeline using GitHub Actions.

### Part 12: Containers

- **Topics**: Docker fundamentals, containerizing apps, and multi-container setups with Docker Compose.
- **Key Project**: Packaging the full-stack app into Docker containers.

### Part 13: Relational Databases

- **Topics**: SQL, relational modeling, PostgreSQL, and database migrations.
- **Key Project**: Integrating PostgreSQL and building relational CRUD functionality.

## Projects

Each part has associated projects to reinforce the concepts taught. You can find the project files inside the respective part folders. Some of the notable projects include:

- **Course Information**: Basic React app to manage course details.
- **Phonebook**: Full-stack application using React, Node.js, and MongoDB.
- **Blog List**: A blog platform with both frontend and backend.
- **Anecdote App**: Redux-based app for managing anecdotes.
- **Book Library**: GraphQL-based app for managing a book library.
